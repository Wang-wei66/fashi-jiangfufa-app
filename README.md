# 法师讲佛法的方法（PWA 网页 App）

> 终极深度版 · 含测试与场景模块

## 在线访问

开启 GitHub Pages 后，通过 `https://<你的用户名>.github.io/<仓库名>/` 直接访问。

## 功能

- **底部 Tab 导航**：首页 · 手法 · 宗派 · 测试 · 场景，五个板块清晰切换
- **十二种讲法手法**：深度展开 + 白话翻译 + 详细案例
- **五宗派操作**：禅、净土、天台、华严、唯识
- **测试中心**：21 组测试、216 道题，逐题即时反馈，进度自动保存
- **场景案例库**：30 个实际说法场景，每个含具体实操（350-470字）
- **移动端优化**：字体可调、暗色模式、安全区域适配
- **PWA 支持**：可添加到手机桌面，像原生 App 一样使用

## 使用方法

1. 在浏览器中打开 `index.html`
2. 手机浏览器中可选「添加到主屏幕」
3. 无需安装，无需联网（首次加载后）

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | PWA 单文件应用（主文件） |
| `原始版.html` | 原始长文档版（含全部内容但不带 App 导航） |

## 技术栈

纯前端单文件 HTML，所有 CSS/JS/数据内联，无依赖，双击即用。
